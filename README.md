# Monte Carlo Retirement Forecast Dashboard

This project models long-term retirement outcomes using Monte Carlo simulation.

The model generates 10,000 simulated market paths to estimate the probability of reaching retirement portfolio targets.

## Features

- 10,000 Monte Carlo simulations
- Bear, Base, and Bull return scenarios
- Probability of reaching $100k, $250k, $500k, and $1M
- Visualization of simulated portfolio growth
- Runs locally in any browser

## Model Overview

The simulation assumes annual returns drawn from historical market distributions and projects portfolio growth from 2026–2050.

Each run represents one potential market trajectory.
