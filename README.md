# Monte Carlo Retirement Forecast Dashboard

A financial forecasting dashboard that simulates **10,000 possible market outcomes** to estimate long-term retirement portfolio growth.

This project uses **Monte Carlo simulation** to model potential investment paths from **2026 to 2050**, helping visualize the range of possible retirement outcomes under different market conditions.

---

## Simulation Fan Chart

The fan chart below illustrates the distribution of portfolio outcomes across thousands of simulated market paths.

![Simulation Fan Chart](simulation-fan-chart.jpg)

The **median line** represents the most likely portfolio growth trajectory.

The shaded area represents the **10th–90th percentile range**, showing the uncertainty of long-term market outcomes.

---

## Key Features

• 10,000 Monte Carlo simulations  
• Bear, Base, and Bull market scenarios  
• Probability tracker for retirement targets  
• Visual fan chart showing uncertainty range  
• Historical return assumptions built into the model  
• Runs locally in any browser

---

## Simulation Assumptions

Base Scenario

• Expected annual return (μ): **5.8%**  
• Market volatility (σ): **16%**

Projection Period

• **2026 – 2050**

The simulation generates thousands of possible return paths to estimate the probability of reaching different retirement targets.
